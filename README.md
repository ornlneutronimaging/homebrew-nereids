# homebrew-nereids

Homebrew tap for [NEREIDS](https://github.com/ornlneutronimaging/NEREIDS) — Neutron Resonance Resolved Imaging Data Analysis System.

## Install

```bash
brew install --cask ornlneutronimaging/nereids/nereids
```

## How it works

The `Casks/nereids.rb` file is updated automatically by CI on each
[NEREIDS release](https://github.com/ornlneutronimaging/NEREIDS/releases).
